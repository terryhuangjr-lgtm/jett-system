WHAT THIS DOES
───────────────────────────────────────────────────────────────

Replaces constantly re-reading markdown files with fast database queries.

TOKEN SAVINGS:
  • Reading markdown: 2-5k tokens per file
  • Database query: 200-500 tokens
  • SAVINGS: 80-95% reduction!

SPEED:
  • Markdown: Must read entire file
  • Database: Instant indexed queries
  • 10-100x faster!

QUICK USAGE - FOR JETT (PYTHON)
───────────────────────────────────────────────────────────────

# Initialize
from jett_db import get_db
db = get_db()

# Find athletes
athletes = db.search_athletes(sport="Basketball", min_value=1000000)
for athlete in athletes:
    print(f"{athlete['name']}: ${athlete['contract_value']:,.0f}")

# Get pending content
content = db.get_pending_content()

# Search research
research = db.search_research(query="NIL", tag="2024")

# Manage tasks
tasks = db.get_tasks_by_status("pending")

# Get stats
stats = db.get_stats()

───────────────────────────────────────────────────────────────
WHEN TO USE DATABASE VS FILES
───────────────────────────────────────────────────────────────

USE DATABASE:
  ✓ Querying/searching data
  ✓ Filtering and sorting
  ✓ Getting statistics
  ✓ Repeated access to same data
  ✓ Token efficiency matters

USE MARKDOWN FILES:
  ✓ Long-form reading
  ✓ Rich formatting needed
  ✓ One-time full document read

BEST PRACTICE:
  • Query database first (fast, cheap)
  • Read markdown only if you need full context
  • Link using source_file field

───────────────────────────────────────────────────────────────
TOKEN SAVINGS EXAMPLE
───────────────────────────────────────────────────────────────

Query: "Find basketball players with contracts over $5M"

BEFORE (Markdown):
  1. List directory: 500 tokens
  2. Read 10 files @ 3k each: 30,000 tokens
  3. Parse and filter: 1,000 tokens
  TOTAL: 31,500 tokens ($0.09)

AFTER (Database):
  1. SQL query: 200 tokens
  2. Parse results: 200 tokens
  TOTAL: 400 tokens ($0.001)

SAVINGS: 98.7% reduction!

For 20 queries/day:
  • Before: 630,000 tokens/day ($1.89/day = $57/month)
  • After: 8,000 tokens/day ($0.02/day = $0.60/month)
  • MONTHLY SAVINGS: $56!
