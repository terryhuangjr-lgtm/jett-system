## How Content Flows

**Research → Ideas → Verification → Tweets → Deployment**

1. **Research phase:** Find raw data with sources
2. **Content generator:** Create tweet concepts from research
3. **Verification:** Run through validator (sources, calculations, URLs)
4. **Tweet creation:** Format for 280 chars, multiple options
5. **Deployment:** Post to #21msports for Terry's review

---

## Quality Checklist

Every research entry MUST have:
✅ Source URL (web link)
✅ Date verified
✅ Key data points (numbers, dates, names)
✅ Content angle (how to use it)
✅ Pillar categorization
✅ No fabrication (if you can't verify, don't log it)

Every content idea MUST have:
✅ Clear hook (attention-grabbing opening)
✅ Verified data (from research)
✅ Multiple tweet options
✅ Character count under 280
✅ Pillar classification
✅ Source attribution
