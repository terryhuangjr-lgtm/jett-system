# 21M Sports Content Generator

Automated content generator for @21MSports - analyzing sports business through a Bitcoin standard.

## Features

- **BTC Price Conversion**: Convert any USD contract value to Bitcoin terms (historical & current)
- **Real-time Sports Data**: Pull current signings, contracts, and news
- **Multiple Post Options**: Generate 3-5 ready-to-post tweet variations
- **Template Library**: Pre-built templates for all content pillars
- **Fact-Checked Data**: All numbers verified with sources

## Quick Start

```bash
# Install dependencies
npm install

# Generate content
npm run generate

# Or run directly
node index.js --theme contract-comparison
node index.js --theme fiat-friday
node index.js --theme athlete-story
```

## Content Pillars

1. **Contracts in BTC Terms** - Historical deals valued in today's BTC
2. **Athlete Wealth Destruction** - Time preference lessons from sports
3. **Sports Business Economics** - TV deals, stadiums, franchise sales in BTC
4. **Macro + Sports** - Fed decisions impact on sports economics

## Weekly Schedule

- **Monday**: 21M Monday (breaking news in BTC terms)
- **Thursday**: Timechain Thursday (historical moments)
- **Friday**: Fiat Debasement Friday (era comparisons)
- **Saturday**: Sat Stacking Saturday (educational content)
- **Sunday**: Sound Money Sunday (athlete wealth stories)

## Usage

The generator will output 3-5 tweet options with:
- Fact-checked data
- BTC conversions
- Character count
- Source links
- Suggested posting time

Choose your favorite and post!

## Configuration

Edit `config.json` to customize:
- Sports teams to follow
- Post frequency
- Data sources
- Template preferences
